45   * <p><strong>Intelligence Purpose:</strong></p>
46   * <ul>
47   *   <li><strong>Temporal Analysis Framework:</strong> All-time politician productivity comparison and analysis</li>
48   *   <li><strong>Pattern Recognition:</strong> Identifies document activity profiles (Party-focused, Individual-focused, etc.)</li>
49   *   <li><strong>Performance Monitoring:</strong> Tracks overall politician legislative productivity metrics</li>
50   *   <li><strong>Strategic Analysis:</strong> Analyzes collaboration patterns and activity levels across careers</li>
51   * </ul>
52   * 
53   * <p><strong>Key Metrics:</strong></p>
54   * <ul>
55   *   <li>Total documents and document type breakdown (motions, propositions)</li>
56   *   <li>Motion type classification (party, individual, committee, multi-party)</li>
57   *   <li>Activity timeline (first/last document, years active, docs per year)</li>
58   *   <li>Activity profile classification (Party-focused/Individual-focused/Committee-focused/Mixed)</li>
59   *   <li>Activity level classification (Very High/High/Medium/Low/Inactive)</li>
60   *   <li>Collaboration percentage based on multi-party motions</li>
