1   /**
2    * CIA (Citizen Intelligence Agency) Web Module.
3    *
4    * <p>
5    * This module delivers the Vaadin-based user interface, enabling users to
6    * explore political data, perform analyses, and manage administrative
7    * operations.
8    * </p>
9    *
10   * <p>
11   * Key Features:
12   * </p>
13   * <ul>
14   * <li>Vaadin UI for interactive dashboards</li>
15   * <li>Spring Security integration</li>
16   * <li>Responsive layouts and charting</li>
17   * </ul>
18   *
19   * <p>
20   * Technologies / Integrations:
21   * </p>
22   * <ul>
23   * <li>Vaadin for UI components</li>
24   * <li>Spring Framework for security and backend integration</li>
25   * <li>Third-party libraries for PDF viewing, charting, and user-agent
26   * parsing</li>
27   * </ul>
28   *
29   * <p>
30   * This module implements the web interface for the CIA application using Vaadin
31   * framework.
32   * It provides a comprehensive user interface for monitoring political figures,
33   * institutions,
34   * and analyzing political/financial trends.
35   * </p>
36   *
37   * <p>
38   * The module contains views and components for:
39   * </p>
40   * <ul>
41   * <li>Administrative operations and system management</li>
42   * <li>Parliament data visualization and analysis</li>
43   * <li>Political party information and statistics</li>
44   * <li>Government body and committee tracking</li>
45   * <li>Document management and ballot tracking</li>
46   * <li>Politician profiles and activities</li>
47   * </ul>
48   *
49   * <p>
50   * Key Features:
51   * </p>
52   * <ul>
53   * <li>Responsive web interface using Vaadin framework</li>
54   * <li>Security integration with Spring Security</li>
55   * <li>Data visualization with charts and grids</li>
56   * <li>PDF viewing capabilities</li>
57   * <li>QR code generation</li>
58   * <li>User agent parsing and analytics</li>
59   * </ul>
60   *
61   * <p>
62   * The module is organized into the following main packages:
63   * </p>
64   * <ul>
65   * <li>application - Core application infrastructure</li>
66   * <li>views.admin.* - Administrative interface components</li>
67   * <li>views.user.* - User-facing interface components</li>
68   * <li>views.common.* - Shared UI components and utilities</li>
69   * </ul>
70   *
71   * <p>
72   * This module requires and integrates with various CIA service modules and
73   * external
74   * data sources including Riksdagen (Swedish Parliament), Val (Elections), and
75   * World Bank data.
76   * </p>
77   *
78   * @see com.hack23.cia.service.api
79   * @see com.hack23.cia.web.widgets
80   */
